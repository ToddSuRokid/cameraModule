Rokid Camera

The camera app for Rokid Glass can take pictures and videos. Saves the file to `sdcard\DCIM\` folder. The user can swipe between picture and video mode by swiping on the touch bar.


## Android API:

Photo:
```
CameraDevice
```

Video:
```
MediaRecorder
```
